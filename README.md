# Development
This project is being developed from scratch (minus a couple of low level libraries) on Linux in Neovim support for Windows is planned to be maintained.

# Build System

## Linux
```bash
mkdir build
cd build
cmake ../
make
```

## Windows
Use Visual Studio CMake integration.
